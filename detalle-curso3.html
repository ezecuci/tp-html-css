Maquetación de layouts modernos con Flexbox y CSS Grid

  • Valor: $20 USD
  • Tiempo de dedicación necesario: 40 hs
  • Descripción del curso: Dominá la creación de interfaces modernas y responsivas con Flexbox y CSS Grid. Vas a repasar el box model y buenas prácticas de accesibilidad, aprender patrones reales (navbars, cards, galerías, dashboards) y combinar Flexbox + Grid para lograr layouts fluidos, escalables y fáciles de mantener. Incluye tips de naming, organización de CSS y estrategias para breakpoints limpios.
  • Requisitos Previos: Conocimientos básicos de HTML semántico y CSS (selectores, propiedades y modelo de caja), manejo de media queries y unidades relativas (%, rem, fr), y familiaridad con herramientas de desarrollo del navegador (inspector y responsive mode). Se recomienda entender principios básicos de accesibilidad y organización de estilos.
